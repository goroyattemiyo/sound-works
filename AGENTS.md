# SOUND WORKS Agent Guidelines

- Use a single HTML file structure.
- Treat `index.html` as the main application file.
- Do not add external libraries.
- Use vanilla JavaScript only.
- Use the Web Audio API for audio features.
- Keep CSS and JavaScript inside `index.html`.
- Do not break existing code.
- Implement changes with small diffs.
- Prioritize readability.
- Preserve the retro chiptune-inspired world and tone.
